Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside the-farm-house folder.

Pages

    • index.html is for default home page with Text & background image slider
    • index-with-fixed-bgimage-and-fixed-text.html is for fixed background image & fixed test
    • index-with-image-slider-with-fixed-text.html is for background image slider with fixed text
    • index-with-text-slider-and-fixed-bgimage.html is for text slider with fixed background image
    • index-with-video-background-and-text-slider.html is for youtube video background with text slider
    • index-with-video-background-with-fixed-text.html is for youtube video background with fixed text
    • index-with-parallax-background.html is for parallax background
    • index-without-animation-effects.html is for without animation effects
